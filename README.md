# BGU Spark

## מה חדש

- **VPL מסומן כהושלם אוטומטית:** לאחר הגשה ב־VPL, המשימה מסומנת כ־Done ולא מופיעה יותר ברשימת המשימות הפעילות.
- **תמיכה ב־Dark Mode:** ניתן לבחור מצב תצוגה בהגדרות התוסף:
  - כמו המערכת
  - בהיר
  - כהה

## הפעלת Dark Mode

1. פתחו את דף ההגדרות של התוסף.
2. תחת **העדפות** בחרו את **ערכת נושא** הרצויה.
3. הבחירה נשמרת אוטומטית ומוחלת על חלון התוסף וגם על רכיבי העיצוב שהתוסף מזריק במודל.
